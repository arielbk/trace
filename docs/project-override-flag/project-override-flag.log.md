## `create-with-project-flag` — 2026-06-06

**Status:** done
**Summary:** Added `resolveProjectRootArg(projectArg, cwd)` to `@trace/core` (in `packages/core/src/project-root.ts`, re-exported from `index.ts`): with no flag it delegates to today's `resolveProjectRoot(cwd)`; with a flag it resolves the override against cwd (when relative), hard-errors on a nonexistent path naming it, then walks up from the override to its `.git` root.
